Transformaciones

Las transformaciones pueden ser de dos tipos, 2d y 3d. Para ambas tenemos una sola propiedad que recibe funciones como valor

Transformaciones 2D

transform:

  • translate (x | x, y) | scaleX() | scaleY()
    • Valores positivos -> derecha abajo
    • valores negativos -> izquierda arriba
  • scale (x = y | x, y) | scaleX() | scaleY()
  • skew (x = y | x, y) | skewX(angle) | skewY(angle)
  • rotate (angle) -> Tenemos 4 formas de dar angulos en CSS
    • deg(360deg) -> grados sexadecimales
    • grad (400grad) -> unidad de medida de angulos planos
    • rad (6.2832rad) -> unidad de angulo en el plano de SI
    • turn (1turn) -> turn es el numero de vueltas
  • Cuando usamos angulos:
    • valores positivos -> sentido horario
    • valores negativos -> sentido antihorario
