Transformaciones

Las transformaciones pueden ser de dos tipos, 2d y 3d. Para ambas tenemos una sola propiedad que recibe funciones como valor

Transformaciones 2D

transform:

  • translate (x | x, y) | scaleX() | scaleY()
    • Valores positivos -> derecha abajo
    • valores negativos -> izquierda arriba
  • scale (x = y | x, y) | scaleX() | scaleY()ß
  • rotate (angle) -> Tenemos 4 formas de dar angulos en CSS
    • deg(360deg) -> grados sexadecimales
    • grad (400grad) -> unidad de medida de angulos planos
    • rad (6.2832rad) -> unidad de angulo en el plano de SI
    • turn (1turn) -> turn es el numero de vueltas
  • Cuando usamos angulos:
    • valores positivos -> sentido horario
    • valores negativos -> sentido antihorario
Todos las cajas tienen las siguientes clases

Ejemplo 1

En este ejemplo se ocupó el translateX con un 25% de translacion, mas una transicion que apunta al transform

                
                    #box-1 {
                        transition: transform 0.5s;
                    }
                    
                    #box-1:hover {
                        transform: translateX(25%)
                    }

Ejemplo 2 - Valores positivos y negativos en translate()

Tambien se puede usar el translate para mover los dos ejes en la misma linea:

                
                    /* para box 2 valores positivos*/
                    #box-2 {
                        transition: transform 0.5s;
                    }
                    
                    #box-2:hover {
                        transform: translate(25%,35%)
                    }


                    /* para box 3 valores negativos*/
                    #box-3 {
                        transition: transform 1s;
                    }
                    
                    #box-3:hover {
                        transform: translate(-50%, -35%)
                    }

Ejemplo 4 rotate

puedes rotar los elementos segun la unidad que estes usando, positivos es para el sentido del las manecillas del reloj y negativo es al contrario

            
                #box-6, #box-7 {
                    transition: transform 5s;
                    text-align: center;
                }
                
                #box-6:hover {
                    transform: rotate(2turn)
                }
                
                #box-7:hover {
                    transform: rotate(-45deg)
